% ==============================================================================
\begin{frame}{Актуальность работы}
В работе рассматривается задача поиска неточных протяженных повторов в биологических последовательностях. Важность поиска повторяющихся элементов 
обусловлена биологической ролью повторов в функционировании организмов.

\bigskip

Нахождение повторов позволяет решать многие задачи:
\begin{itemize}
    \item Определение родства групп организмов на генном уровне.
    \item Диагностика генетических заболеваний.
    \item и другие
\end{itemize}
\end{frame}
% ==============================================================================


% ==============================================================================
\begin{frame}{Мотивация постановки задачи}
Мотивация постановки задачи связана с растущими объемами генетических данных и
необходимостью проведения многократных быстрых вычислительных экспериментов.

\bigskip

В данной работе предлагается и исследуется возможность суперкомпьютерной реализации
метода поиска повторов в~биологических последовательностях.
\end{frame}
% ==============================================================================


% ==============================================================================
\begin{frame}{Рассматриваемый метод}
В дипломной работе рассматривается метод поиска протяженных размытых повторов,
разработанный коллективом сотрудников кафедры математических методов
прогнозирования ВМК МГУ и института математических проблем в биологии Российской академии
наук (Пущино).

\bigskip

Дипломная работа выполнялась в~тесном сотрудничестве с~авторами метода.
\footnote{Тетуев .Р.К., Назипова Н.Н., Панкратов А.Н., Дедус Ф.Ф. \\
Поиск мегасателлитных повторов в геномах эукариот по оценке осцилляций
кривых GC- содержания// Математическая биология и биоинформатика, 2010, Т. 5, № 1.}
%\label{footnote1}
\end{frame}

% ==============================================================================
\begin{frame}{Задачи дипломной работы}
\begin{itemize}
    \item Разработка и реализация параллельного алгоритма поиска повторов,
          ориентированного на суперкомпьютерную реализацию: большой объем
          обрабатываемых данных (размер данных порядка 1 ГБ),
          время обработки в~пределах 1 часа.
    \item Исследование эффективности использования графических процессоров для решения поставленной задачи.
    \item Анализ маcштабируемости разработанного алгоритма на~примере задачи сравнения конкретных геномов
    \item Разработка графического интерфейса для работы пользователя на локальной и удаленных системах.
\end{itemize}
\end{frame}

Вычислительные платформы для решения поставленных задач:
\begin{itemize}
    \item суперкомпьютеры Ломоносов, BlueGene/P
    \item 2-х процессорная рабочая станция на базе 4-х ядерного процессора

Предложенные платформы обладают различной архитектурой и позволяют исследовать
разрабатываемый алгоритм с учетом их особенностей.

\begin{frame}{Исследуемые последовательности}

Необходимо найти \textbf{протяженные неточные} повторы.

\begin{itemize}
    \item протяженные - длинные повторы (от 1000 эл.)
