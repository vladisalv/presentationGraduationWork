\begin{frame}{Схема параллельного алгоритма поиска повторов}
На вход алгоритму подаются:
    \begin{itemize}
        \item Две биологические последовательности (массив символов четырехбуквенного алфавита A, T, G, C)
        \begin{itemize}
            \item Как один из вариантов - одна последовательность, т.е. последовательность сравнивается сама с собой
        \end{itemize}
        \item Параметры алгоритма
            \begin{itemize}
                \item Длина скользящего окна профилировки
                \item Длина скользящего окна построения спекторов
                \item Шаг сдвига окна построения спекторов
                \item $\varepsilon$ - точность сравнения спекторов
                \item Параметры фильтрации найденных повторов
            \end{itemize}
    \end{itemize}
\end{frame}
