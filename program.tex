\begin{frame}{Структура программной реализации параллельного алгоритма}
Параллельная реализация программы выполнена на С++ с использованием технологии
параллельного программирования MPI и технологии CUDA-C для использования
графических ускорителей Nvidia.

\bigskip

Взаимодействие с MPI и CUDA происходит с помощью отдельных классов, что позволяет
при желании откомпилировать программу без использования этих технологий.
\end{frame}


\begin{frame}{Структура программной реализации параллельного алгоритма}
Структура алгоритма позволяет выделить каждый этап в отдельный модуль. Таким образом
работу программы можно начать с любого этапа и закончить на любом этапе.

\bigskip

В стуктуре классов можно выделить следующие группы:
\begin{itemize}
    \item Классы сущности
    \item Классы вычислений
    \item Классы управления
\end{itemize}

\bigskip

Объем кода всего проекта (оснавная программа + GUI) составляет 6 тыс. строк.
\end{frame}
